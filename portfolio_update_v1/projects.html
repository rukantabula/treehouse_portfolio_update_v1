Trend Detector, Android App

This app uses artificial intelligence to recognise patterns from images taken or uploaded by a user. The app will then return results with similar images, brands, prices and websites where they can be found.

While building this app i used Photoshop for designing the User Interface and Java/Android Studio for developing the app. For storage I used Firebase Realtime database.

I have integrated the app with the open source API that does the image processing. The app sends images to the API and get back image's meta data as a response to the call. For this case the meta data include details like similar clothes and accessories that resembles the image uploaded, price, description and websites where these items can be purchased.
